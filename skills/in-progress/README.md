# Em Progresso

Skills que ainda estão sendo desenvolvidas. Elas não estão prontas para entregar — espere arestas, mudanças quebrando compatibilidade e experimentos abandonados. Estão excluídas do plugin e do README de nível superior até graduarem para um bucket estável.

- **[review](./review/SKILL.md)** — Revisa mudanças desde um ponto fixo em dois eixos paralelos: **Standards** (o diff segue os padrões de código do repo?) e **Spec** (o diff implementa fielmente a issue/PRD originadora?).
- **[writing-beats](./writing-beats/SKILL.md)** — Modela um artigo como uma jornada de beats, no estilo escolha-sua-aventura. Escolha um beat inicial, escreva apenas aquele beat, depois pivote para o próximo, até o artigo chegar a um fim natural.
- **[writing-fragments](./writing-fragments/SKILL.md)** — Sessão de grilling que minera fragmentos de você — pepitas heterogêneas de escrita — e os anexa a um único documento como matéria-prima para um futuro artigo.
- **[writing-shape](./writing-shape/SKILL.md)** — Pega um arquivo markdown de matéria-prima e o molda em um artigo parágrafo por parágrafo, argumentando escolhas de formato a cada passo.
